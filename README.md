# Analyse de Sentiments des Commentaires YouTube 🎯

Ce projet a pour objectif de classifier les commentaires YouTube en sentiments **positifs** ou **négatifs** en utilisant le **Traitement Automatique du Langage Naturel (NLP)** et des modèles de **Machine Learning**.

## 🧠 Technologies utilisées

- Python
- Pandas, NLTK, Scikit-learn
- Matplotlib, Seaborn
- TF-IDF Vectorizer
- Support Vector Machine (SVM)

## 📁 Structure du projet

- `notebook.ipynb` : le notebook principal contenant le code, le prétraitement, la visualisation et la classification.
- `requirements.txt` : les dépendances nécessaires pour exécuter le projet.
